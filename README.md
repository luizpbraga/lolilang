Loli is an interpreter that sometimes does not know how to interpret.


##### code example
```go
const {
    ONE = 1
    TWO = 2
}

var foo = fn(x, y) {
    var hello = "Hello Loli"
    return if x > y { x } else { hello }
}

var v = { 1, true, "2" , false, 4 }

v[0] += 1

print( foo(1,2), v[0], {1,2,3} , ONE + 1 == TWO, "Tchau!".len )
```
